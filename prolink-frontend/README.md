## Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Backend Configuration

The frontend now falls back to the hosted ProLink backend automatically, so public pages, signup, login, and jobs should work even if you are only running the frontend locally.

If you want to point the app to a different backend, set one of these environment variables before starting Next.js:

```bash
NEXT_PUBLIC_API_BASE_URL=https://your-backend.example.com/api
# or
NEXT_PUBLIC_API_URL=https://your-backend.example.com/api
NEXT_PUBLIC_SOCKET_URL=https://your-backend.example.com
```

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
